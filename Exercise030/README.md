# Exercise 030
## Spec
Course grading
Here is the grading for this course:

| Percent  | Grade        |
|----------|--------------|
| < 0      | Impossible   |
| 0 - 49   | Fail         |
| 50 - 59  | 1            |
| 60 - 69  | 2            |
| 70 - 79  | 3            |
| 80 - 89  | 4            |
| 90 - 100 | 5            |
| > 100    | Outstanding! |


Create a program, which asks the user for their percent, and gives them their score.

## Explanation
A Simple exercise with a overengineered solution. Helped understand C# and the syntax.
